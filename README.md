# 清莲引 - AI驱动廉洁素养实训系统

> 新时代大学生廉洁教育互动实训平台

本作品以"贪吃蛇"为交互载体，融合**多智能体 AI 对抗**、**机器学习风险评估**、**行为聚类分析**、**NLP 知识赋能**四大 AI 技术，构建"**诱惑 — 抉择 — 评估 — 教育**"的完整闭环。

## 四大 AI 技术亮点

### 🤖 多智能体 AI 对抗系统
- **独立化 AI 蛇**：页面加载即启动，在红线右侧自主游荡觅食
- **三种可切换人格**：
  - 🔵 **谨慎型**：红线边缘巡逻，只吃安全区词汇
  - 🟡 **贪婪型**：抢玩家附近的词，直冲高价值稀有词
  - 🔴 **疯狂型**：无视红线，主动撞击玩家
- **视野分层决策**：近程(3格)抢词 / 中程(3-8格)找稀有 / 远程(8格外)巡逻制造压迫感
- **可解释 AI 决策面板**：实时显示状态、目标、风险计算过程

### 📊 机器学习风险评估
- **廉洁风险指数**：实时计算 0-100% 风险等级
- **行为画像分析**：莲心稳固 / 边缘试探 / 渐入迷途 / 危险倾向
- **多维度特征追踪**：越界次数、廉洁词收集、负面词摄入、安全区停留时间

### 🎯 行为聚类分析
- **廉洁修行支线任务**：基础/进阶/挑战三种难度
  - 洁身自好：不迟到任何负面词通关
  - 慧眼识珠：连续5次比AI先吃到稀有词
  - 深入虎穴：在红线右侧停留超过30秒并安全返回
- **成就系统**：莲华·不染、边缘行者、背水一战

### 📚 NLP 知识赋能
- **快问快答**：关卡间3秒限时廉政知识问答
- **典故小卡片**：稀有词触发古风廉洁故事卡片
- **警示语录**：AI越线时飘过"一念之差，满盘皆输"等警示语
- **AI 廉洁导师**：三级响应引擎（AC自动机+TF-IDF+大模型）

## 核心玩法

### 游戏机制
- **红线机制**：蛇越过红线吃红色诱惑点，蛇身崩解、扣分、缩短
- **廉洁词云**：收集"清正""自律""慎独"等词汇
- **负面词惩罚**：吃到"贪""贿""腐"扣分并缩短蛇身
- **五重莲境关卡**：莲心初醒→莲骨渐成→莲叶田田→莲茎挺拔→莲华绽放

### AI 心魔对抗
- **碰撞后果**：
  - 玩家撞AI蛇 → 扣15廉洁值 + 1秒迷惑效果
  - AI撞玩家蛇头 → AI被弹回红线外停顿2秒
- **正气道具**：免疫AI撞击3秒，撞到AI将其弹回出生点

### 随机事件
- 🌀 廉政风暴：所有负面词消失10秒
- 🎁 糖衣炮弹：3个绿色词暂时变红
- ✨ 双倍积分：所有词分数翻倍30秒
- 🌫️ 迷雾笼罩：红线暂时消失10秒
- 💥 心魔暴走：AI进入疯狂型1分钟

## 技术栈

- **前端**: HTML5 Canvas + 原生JavaScript (ES6+)
- **AI 对抗**: 有限状态机多智能体决策系统
- **风险评估**: TensorFlow.js 浏览器端机器学习
- **知识问答**: NLP 语义匹配 + 大模型生成
- **后端**: Node.js + Python Flask

## 快速开始

```bash
# 安装依赖
npm install

# 启动前端服务器
node server.js
# 访问 http://localhost:3000

# 启动后端 AI 服务器（可选）
cd backend
python doubao_api_server.py
```

### 🔑 API Key 配置（部署后）

1. **获取 API Key**：在 [豆包开放平台](https://www.doubao.com/) 注册并获取
2. **本地开发**：复制 `backend/doubao_api_server.example.py` 为 `doubao_api_server.py`，替换 `API_KEY` 变量
3. **线上部署**：设置环境变量 `DOUBAO_API_KEY`

### 📝 示例配置

```python
# backend/doubao_api_server.py
API_KEY = "your-actual-api-key-from-doubao"
```

### ⚠️ 安全注意事项

- **切勿提交包含真实 API Key 的文件**
- 使用环境变量管理敏感信息
- 在前端使用代理服务器转发 API 请求
- 定期轮换 API Key
