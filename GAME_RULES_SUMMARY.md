# 🪷 清莲引 · 浮世贪吃蛇 - 规则摘要
## 一、核心玩法
| 项目 | 说明 |
|------|------|
| **胜利条件** | 完成莲境三关，收集词汇，三境圆满 |
| **单关胜利** | 收集8个廉洁词汇进入下一关 |
| **失败条件** | 越界3次 / 廉洁值0 / 撞墙或自噬 |

## 二、莲境三关系统
| 关卡 | 红线位置 | 难度 | 隐喻 |
|------|---------|------|------|
| 第1境 · 莲心 | 50%中线 | 入门 | 初始廉洁空间宽裕 |
| 第2境 · 莲骨 | 42%偏左 | 进阶 | 地位越高，约束越紧 |
| 第3境 · 莲华 | 34%最左 | 终极 | 考验自律，成就升华 |

## 三、食物系统
| 类型 | 效果 | 说明 |
|------|------|------|
| 廉洁词（绿） | +分 +廉洁值 | 安全区优先刷新 |
| 稀有词（金边绿底） | +高分 +慎独技能 | 仅右侧高风险区刷新 |
| 负面词（红） | -分 -廉洁值 | 蛇身缩短 |
| 诱惑点（金$） | -分 -廉洁值 | 红线附近随机刷新 |

## 四、AI动态难度调节
### 风险评分系统（每10秒计算一次）
| 评分范围 | 等级 | 安全区绿色词 | 红线收缩速度 |
|---------|------|-------------|------------|
| 0-30% | 🟢 低风险 | 正常（4-2个） | 40秒一次 |
| 31-60% | 🟡 中风险 | 减少（3-1个） | 22秒一次 |
| 61-100% | 🔴 高风险 | 仅保底1个 | 15秒一次 |

### 触发负面词联动
- **越界后**：安全区随机生成负面词（50%概率）
- **AI抢词后**：被抢位置高概率生成负面词
- **隐喻**：投机取巧的后果会污染整个环境

## 五、AI·贪欲蛇系统
| 模式 | 概率 | 行为特点 | 隐喻 |
|------|------|---------|------|
| 保守模式 | 40% | 只在安全区觅食 | 坚守底线，谨慎守规 |
| 贪婪模式 | 30% | 优先追求稀有词，不惜越界 | 贪欲无限，见利忘义 |
| 躲避模式 | 30% | 寻找离玩家最远的食物 | 差异化竞争，博弈智慧 |

## 六、慎独技能系统
| 项目 | 说明 |
|------|------|
| 初始次数 | 2次 |
| 充能方式 | 每吃到1个稀有词，技能+1 |
| 使用效果 | 蛇瞬间传送回安全区，重置越界状态 |
| 哲学隐喻 | 慎独，在无人看见时依然坚守底线；即使犯错，也有悬崖勒马的机会 |

## 七、廉洁档案系统
### 自动存档（localStorage）
- 个人记录：最高分、总游戏次数、累计越界次数
- 成就徽章：解锁的所有成就
- 近期记录：最近10次游戏详情
- 风险画像：长期风险模式分析

### 风险画像类型
| 类型 | 特征 | 建议 |
|------|------|------|
| 莲心稳固型 | 平均越界<0.5次/局 | 定力超群，继续保持 |
| 边缘游走型 | 0.5-1.5次/局 | 偶尔试探，需要警觉 |
| 边界试探型 | >1.5次/局 | 需要时刻警醒，避免滑坡 |

## 八、AI导师五重角色
| 角色 | 核心任务 | 数据依据 |
|------|---------|---------|
| 素养评估师 | 解读雷达图，分析认知偏差 | 游戏数据+雷达图 |
| 个性化教学助手 | 薄弱词汇讲解，出变式题 | 语义判定日志+RAG |
| 评估报告解读官 | 分数转评语，有同理心 | 统计数据+画像 |
| 廉洁知识咨询师 | 回答开放式问题 | 知识库+典故 |
| 学习路径规划师 | 推荐学习计划 | 能力评估+进度 |

## 九、技术亮点
### AI技术创新点
1. **RAG增强生成**：本地知识库+向量检索，提供精准廉洁知识解答
2. **对话上下文管理**：多轮对话记忆，理解连续语境
3. **认知偏差检测**：识别玩家心理模式，提供针对性指导
4. **自适应学习路径**：根据能力短板智能推荐学习模块
5. **后端安全代理**：API Key零暴露，保障系统安全

### 核心创新机制
| 设计 | 创新点 | 隐喻 |
|------|--------|------|
| 莲境三关 | 渐进式难度 + 红线左移 | 地位越高，越需自律 |
| AI动态难度 | 风险评分驱动 + 个性化调整 | 廉洁信用体系，越自律越顺利 |
| 慎独技能 | 吃稀有词充能 + 悬崖勒马 | 犯错仍有改过机会 |
| 廉洁档案 | localStorage持久化 + 风险画像 | 长期行为分析，持续警醒 |
| AI导师 | RAG + 对话记忆 + 五重角色 | 智能引导，因材施教 |

## 十、快速开始
```bash
# 启动游戏服务器
cd clean-handbook-game
node server.js
# 访问 http://localhost:3000
```

### 操作说明
- 移动：方向键 或 WASD
- 暂停：空格键
- AI导师：点击右侧AI导师按钮
- 查看档案：点击右上角评估报告
