### 4.3 后端代理实现 (api/doubao.js)

```javascript
export async function POST(request) {
  const { prompt, options = {} } = await request.json();
  
  // API Key从环境变量获取，前端零敏感信息
  const apiKey = process.env.DOUBAO_API_KEY;
  const modelId = process.env.DOUBAO_MODEL_ID || 'doubao-seed-1-6-250615';
  
  const response = await fetch('https://ark.cn-beijing.volces.com/api/v3/chat/completions', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${apiKey}`  // Key仅在服务端使用
    },
    body: JSON.stringify({
      model: modelId,
      messages: [{ role: 'user', content: prompt }],
      temperature: options.temperature || 0.8,
      max_tokens: options.max_tokens || 1024
    })
  });
  
  return NextResponse.json({ success: true, data: result });
}
```

### 5.3 检索算法

```javascript
retrieveRelevantKnowledge(query, topK = 3) {
    // 相关性评分算法
    if (query包含概念本身) score += 0.5;
    if (query包含相关词汇) score += 0.3;
    if (query包含例句词汇) score += 0.2;
    if (query包含反义词汇) score += 0.15;

    // 返回topK最相关结果
    return results.sort(by score desc).slice(0, topK);
}
```

- **风险画像**：长期风险模式分析

## 十五、AI·廉洁风险预测（机器学习模块）

### 数据采集
- 越界次数及频率
- 负面词/正面词摄入比例
- 在右侧危险区停留时间占比
- 总游戏时长

### 逻辑回归模型
使用4个特征 + 权重矩阵 + Sigmoid函数输出风险评分：

```
风险评分 = Sigmoid( -0.4 + 0.45×越界因子 + 0.25×负面词比例 + 0.2×危险时间 + (-0.1)×得分 )
```

## 十九、答辩话术指南

### 问题1：AI导师如何保证回答的准确性？
> 我们采用了RAG（检索增强生成）技术。在回答问题前，系统会先从本地知识库检索相关的廉洁概念、典故、法规等资料，然后将这些资料作为上下文提供给大模型。这确保了回答有据可查，而不是凭空生成。

### 问题2：为什么要做后端代理？
> 传统的做法是前端直接调用豆包API，但这会暴露API Key。我们的方案是前端只发送prompt，后端代理使用环境变量中的API Key进行调用。这样API Key永远不会被传输到浏览器端，安全性大大提高。

### 问题3：学习路径是如何智能推荐的？
> 系统会根据玩家在游戏中的表现（雷达图数据）自动识别薄弱维度。例如，如果语义辨别分数低于70分，系统会推荐从"基础词汇认知"开始学习，逐步进阶到"语义关系理解"。这是一个基于规则的自适应推荐系统。

### 问题4：AI蛇的算法有什么亮点？
> AI蛇采用了改进版A*寻路算法，结合三种行为模式（贪婪/保守/躲避）。更重要的是，AI每次模式切换都会增强红线意识和玩家预判，这模拟了"越陷越深"的人性弱点，让玩家在对比中理解自律的重要性。
